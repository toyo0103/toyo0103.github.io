SQL NOT EXISTS

Posted on 2013-05-28

參考
MSDN
結論就是，如果單一欄位要篩選掉就用Not IN 


select 
from [order]
where 序號 not in
(
 select 訂單編號 from orderDetail
)
如果多個欄位要同時符合才篩選掉，就用Not Exists 

select 
from [order] a
where not exists
(
 select 訂單編號 from orderDetail b
 where b.訂單編號 = a.序號 and b.購買日期 = (a.民國年+1911)
)
